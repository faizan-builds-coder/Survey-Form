
# HTML Theory Assignment 1 Week 1

## Q0. What is HTML and what is the difference between HTML and HTML5?

HTML (HyperText Markup Language) is used to create and structure web pages. It defines the structure of content on the web.

HTML5 is the latest version of HTML. It supports new features like audio, video, canvas, and semantic tags, while older HTML versions did not have these features.

---

## Q1. What are semantic HTML tags? Why are they important in web development?

Semantic HTML tags clearly describe their meaning to both the browser and developers.

Examples: `<header>`, `<footer>`, `<article>`, `<section>`

They are important because they improve SEO, make code easier to read, and improve accessibility for screen readers.

---

## Q2. What is the difference between `<div>` and `<span>` tags?

`<div>` is a block-level element used for grouping content and starts on a new line.

`<span>` is an inline element used to style or group small parts of text within a line.

---

## Q3. Explain the difference between block-level elements and inline elements.

Block-level elements take full width and start on a new line (e.g., `<div>`, `<p>`).

Inline elements only take required space and stay in the same line (e.g., `<span>`, `<a>`).

---

## Q4. What is the purpose of the DOCTYPE declaration in HTML?

DOCTYPE tells the browser which version of HTML is being used. It ensures the page is rendered correctly in standard mode.

---

## Q5. What is the difference between id and class attributes?

`id` is unique and used for a single element.

`class` can be used for multiple elements to apply the same styling or behavior.

---

## Q6. How do you create a form in HTML? Which input types are commonly used?

A form is created using the `<form>` tag.

Common input types include text, email, password, number, radio, checkbox, and submit.

---

## Q7. What are meta tags in HTML and why are they used?

Meta tags provide information about the webpage such as character set, description, and responsiveness. They are used for SEO and proper page rendering.

---

## Q8. Explain the purpose of the alt attribute in the `<img>` tag.

The alt attribute provides alternative text if an image fails to load. It also helps accessibility and SEO.

---

## Q9. How do you make an image clickable in HTML?

You can make an image clickable by placing it inside an anchor (`<a>`) tag.

---

## Q10. What is the difference between JPG, PNG, SVG, and WebP image formats in web development?

JPG is used for photos with smaller file size but no transparency.
PNG supports transparency and better quality.
SVG is a vector format that scales without losing quality.
WebP is a modern format that provides high quality with smaller file size.

---

## Q11. What are semantic tags introduced in HTML5 such as `<header>`, `<footer>`, `<section>`, and `<article>`?

These are tags that give meaning to different parts of a webpage.
`<header>` is for top section, `<footer>` is for bottom section, `<section>` is for grouping content, and `<article>` is for independent content.

---

## Q12. What is the difference between `<script>`, async, and defer in HTML?

`<script>` blocks HTML loading until the script runs.
`async` loads script while HTML is parsing and runs immediately when ready.
`defer` loads script in background and runs after HTML is fully parsed.

---

## Q13. How do you embed audio and video in HTML5?

Audio is embedded using `<audio>` tag and video using `<video>` tag with controls.

---

## Q14. What is the difference between relative paths and absolute paths in HTML?

Relative paths are based on the current folder location.
Absolute paths are full URLs or full file paths.

---

## Q15. What are data attributes in HTML (data-*)? Where are they used?

Data attributes store custom data in HTML elements. They are used to pass information to JavaScript.

---

## Q16. What is the purpose of the viewport meta tag in responsive web design?

It makes the website responsive by controlling layout on different screen sizes, especially mobile devices.

---

## Q17. How can you improve SEO using HTML?

SEO can be improved by using semantic tags, proper headings, meta descriptions, alt attributes, and clean HTML structure.

---

## Q18. What are accessibility best practices in HTML?

Use alt text for images, proper labels for forms, semantic tags, and good color contrast for better accessibility.

---

## Q19. What is the difference between `<strong>` vs `<b>` and `<em>` vs `<i>` tags?

`<strong>` shows important text with meaning (SEO friendly), while `<b>` is just bold styling.

`<em>` adds emphasis with meaning, while `<i>` is just italic styling without importance.
